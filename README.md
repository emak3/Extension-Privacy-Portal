# Extension Privacy Portal

Chrome / Edge拡張機能のプライバシーポリシー、配布先、お問い合わせ情報を一箇所で公開する静的サイトです。

## 公開URL

https://extensions.ouma3.org/

## 構成

- `index.html`: 拡張機能一覧
- `policies/`: 各拡張機能のプライバシーポリシー
- `terms/`: 個別の利用規約
- `contact/`: お問い合わせ
- `assets/`: 共通スタイルとアイコン

GitHub Pagesは `.github/workflows/static.yml` から公開します。
